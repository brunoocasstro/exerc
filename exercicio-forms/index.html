    • 2) Responda as perguntas a baixo:

      1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

        2. R) É utilizado para indicar ao navegador o texto que deverá aparecer escrito no campo do formulário.

      2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

        3. R) É usado para indicar quais checkboxes e qual radio button devem estar ligados inicialmente, se necessário.

      3. Na tag <input> para que serve o atributo "value"?

        4. Para definir o valor a ser enviado por este item.

      4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

        5. R) Agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez, você deve utilizar o atributo "name" (nome) nos elementos<input type="radio"> relacionados.

      5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

        6. R) A principal finalidade do elemento <label> é melhorar a acessibilidade e a usabilidade dos formulários na web, fornecendo uma descrição legível para os campos de entrada.

      6. Na tag <option> para que serve o atributo "selected"?

        7. R) É usado em elementos de seleção, como a tag <select>, para especificar qual das opções deve ser selecionada por padrão quando a página é carregada.

      7. Nas tags de formulário para que serve o atributo "required"?

        8. R)Serve para indicar que um campo deve ser preenchido ou selecionado pelo usuário antes que o formulário possa ser enviado.
        Esse atributo é usado para validar os campos obrigatórios em um formulário e garantir que o usuário forneça informações necessárias antes de prosseguir.

      8. Para que serve a tag <textarea>?

        9. R) É usada para criar uma área de texto multi-linhas em um formulário web, onde os usuários podem inserir texto livremente.

      9. Na tag <form > para que serve o atributo "action"?

        10. R)É usado para especificar o URL do servidor ou do script para o qual os dados do formulário serão enviados quando o formulário for submetido pelo usuário.

      10. Na tag <form> para que serve o atributo "method"?

        11. R)Será utilizado para enviar os dados do formulário para o servidor quando o formulário for submetido pelo usuário.

      11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form >?

        12. R) A visibilidade do valor digitado em um campo de senha na barra de endereços é determinada pelo tipo de campo de entrada de senha <input type="password"> e pelo uso apropriado do atributo "method" não afeta isso.

      12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

        13. R) Isso ocorre porque o envio de arquivos é uma operação que envolve a transmissão de dados binários, como imagens, documentos ou outros tipos de arquivos, e o método POST é apropriado para esse tipo de operação.

      13. O que as tags <fieldset> e <legend> produzem?

        14. R)são usadas para criar grupos de controles de formulário e fornecer uma legenda significativa para esses grupos.
        Isso torna os formulários mais organizados, compreensíveis e acessíveis para os usuários, facilitando a navegação e o preenchimento de informações.
